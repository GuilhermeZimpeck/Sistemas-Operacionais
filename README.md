# Projeto 1
**Descrição**

Neste projeto, você irá desenvolver uma versão simplificada dos comandos UNIX **cat**, **grep**, **zip** e **unzip**. Para não haver confusão, utilize os nomes **wcat**, **wgrep**, **wzip** e **wunzip**, respectivamente.

Objetivos:
* Se desenvolver na linguagem de programação c;
* Se (re)familiarizar com a utilização de um shell / terminal / linha de comando UNIX;
* Aprender (como efeito colateral) como utilizar um editor de código, como o visual studio code;
* Aprender um pouco sobre como os comandos UNIX são implementados.

# Projeto 2
**Descrição**

Neste projeto, você construirá um shell Unix simples. O shell é o coração da interface de linha de comando e, portanto, é fundamental para o ambiente de programação Unix/C. Dominar o uso do shell é necessário para se tornar proficiente nesse mundo; saber como o próprio shell é construído é o foco deste projeto.

Objetivos:
* Para se familiarizar ainda mais com o ambiente de programação Linux.
* Para aprender como os processos são criados, destruídos e gerenciados.
* Para obter experiência com as funcionalidades necessárias em shells.
